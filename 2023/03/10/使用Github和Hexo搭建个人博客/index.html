使用Github和Hexo搭建个人博客

发表于 2023-03-10 更新于 2023-03-11

为什么要搭建博客?

  • 好记性不如烂笔头,在互联网时代,面对于如此多的信息输入,我们也应该有输出;对于学习到的知识点也应该总结出来,所以有一个博客可以让你对知识点进行总结,也更加方便我们后期回顾

如何搭建?

  • 对于搭建博客网络上的教程五花八门,用到的技术也不同,我在这里使用的是成本最低的方式:Github+Hexo

开始搭建

  1. 使用到的技术git,nodejs,npm

  2. 首先我们要有一个Github账号,创建一个仓库取名为:账号名.github.io

  3. 安装hexo

    npm install hexo -g

  4. 创建博客目录,切换到当前目录进行初始化

    hexo init

  5. 找到站点配置文件配置

  6. 继续在该博客目录下执行部署命令

    npm install hexo-deployer-git –save

  7. 命令介绍

关于主题

  • 我们可以安装一些主题来美化博客,推荐两种方案:Butterfly和NexT,这里介绍NexT的使用方式

    NexT的使用

    1. 在博客目录下克隆NexT到themes/next文件夹

      git clone https://github.com/theme-next/hexo-theme-next themes/next

    2. 然后在站点配置文件_config.yml配置,将Hexo主题替换为NexT主题

      theme: next

    3. 在NexT里面还有很多配置,可以去官网了解:NexT
